Slots are like placeholders where you can insert contents into the child component from a parent component by using the "slot" element as the content distribution outlet. A slot can be created just like creating a component. Hence, every slot is just like a component, can have `props`, `data`, `created`, `watch`, `mounted` methods and properties. Slots are very useful when you need to use inline templates, instead of having your templates inside the child components.
